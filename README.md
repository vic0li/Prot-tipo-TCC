# Smart ICU Analytics

Protótipo desenvolvido para o TCC:

Desenvolvimento de um Dashboard Inteligente para Monitoramento de Risco Clínico e Operacional em UTI utilizando dados simulados e metodologias de Melhoria Contínua.

## Tecnologias

- Python
- Streamlit
- Pandas
- Plotly
- Power BI

## Funcionalidades

- Monitoramento clínico
- Alertas inteligentes
- Risco de LPP
- Indicadores Lean Healthcare
- Dashboard preditivo
